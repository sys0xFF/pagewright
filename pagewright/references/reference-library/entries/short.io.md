- **niche:** dev-tools
- **mood:** clean-light
- **style:** minimal, mono-type
- **palette:** bg `#FFFFFF` · ink `#16223A` · accent `#21C275` — primary CTA buttons (Get started, Shorten, Create free account), the second-line hero word 'analytics', logo mark, link icon, eyebrow trust text, and feature icon fills

The page replaces the usual dead hero screenshot with the actual tool you came to use.
- **imagery:** Near-zero photography. The hero is an in-product widget card (rounded, soft-shadowed) acting as both demo and visual anchor. Elsewhere: simple two-tone line icons in pale-green rounded squares, a sketchy hand-drawn industry icon set (logistics, retail, tech, food), and embedded Trustpilot star rating. UI-as-imagery rather than illustration or stock.
- **copy:** Plainspoken, benefit-first, jargon-free — leads with the exact category ('URL shortener with analytics') then stacks concrete capabilities (QR codes, A/B tests, API) instead of slogans.

**Takeaways (steal as ideas, don't copy):**
- Make the hero the product: embed a real, working mini-tool (input -> live result) instead of a static screenshot, so visitors get value before scrolling.
- Two-color hero headline: render the differentiating word ('analytics') in the brand green against ink-dark text to telegraph the value prop in one glance.
- Stack hard proof in the hero zone — 'No credit card required' eyebrow + Trustpilot rating + a 4-up benefit strip — to neutralize friction immediately below the fold.
- Use a pale-green icon-chip system (mint rounded squares) to make a dense utility tool feel calm and scannable rather than enterprise-heavy.
