**Takeaways (steal as ideas, don't copy):**
- Float the headline and the primary input directly on a full-bleed lifestyle photo with no card — composition (a darker quadrant of the photo) does the legibility work instead of a UI panel.
- Make the core conversion action a single low-commitment field (just "Enter home address") and de-risk it inline with a triplet of friction-killers: time, obligation, privacy.
- Reserve the lone saturated accent for the submit button only, so the eye is funneled to the one thing you want clicked inside an otherwise warm, neutral frame.
- Lead the proof section with a precise, oddly-specific live counter ("301,457 homeowners") — the un-rounded number reads as a real database query, not marketing rounding.
