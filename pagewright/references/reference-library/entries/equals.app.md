# Equals — https://equals.app

![equals.app](../thumbs/equals.app.jpg)

- **niche:** data
- **mood:** editorial-minimal
- **style:** minimal, mono-type, colorful
- **palette:** bg `#F7F5EF` · ink `#1A1A1A` · accent `#B98FD9` — Filled pill CTA buttons ('Get a demo'), the active state of the email-capture button; deliberately the ONLY saturated color in the chrome — everything else is ink-on-cream
- **type:** display *A high-contrast transitional serif (Times/Georgia-adjacent, likely a refined cut like Tiempos or a Plantin-style face)* · body *Geometric/neo-grotesque sans for subhead, nav, and labels* — Authoritative print-editorial serif meets clean fintech sans — 'a newspaper that does math'
- **sections:** hero › logos › feature-source-of-truth › feature-pipeline-arr › feature-system-not-artifact › feature-managed-warehouse › how-it-works › cta › footer
- **signature:** The hero headline is set in a massive, tightly-leaded book serif (two lines that nearly touch) on warm paper — an unmistakably EDITORIAL/print voice for a data tool, with the only color being two soft pastel gradient bars (mint-left, butter-yellow-right) bleeding off the page edges like margin swatches. Serif + cream + pastel-bleed is the move a generic SaaS template would never make.
- **imagery:** No product screenshots above the fold — imagery is purely typographic plus abstract pastel gradient bands (horizontal stripes of mint/teal and pale yellow) bleeding off both edges of the hero. Logo grid uses monochrome ink-black customer marks. Treatment is restrained, paper-like, almost zine/print rather than glossy 3D or dashboard mockups.
- **copy:** Blunt, confidence-asserting declaratives that lean into anxiety relief — hero: 'AI for the numbers that can't be wrong.' followed by triadic sub ('Build expert workbooks. Trust every answer. Run it everywhere.'); section heads repeat a 'same question, same answer' / 'stop vibe-coding your numbers, start trusting them' trust-vs-doubt frame.

**Takeaways (steal as ideas, don't copy):**
- Set the hero in a big book-weight serif on warm cream paper instead of a sans on white — it instantly reads 'editorial authority,' a strong contrast trick for a trust-driven data/finance product
- Restrict ALL color to one job: here a single lavender pill is the only saturated element in the entire chrome, so every CTA reads as the one thing to click
- Let abstract pastel gradient bars bleed off the page edges (left/right of the hero) as the sole 'imagery' — color and texture without committing to a product screenshot or 3D render
- Write section headings as paired trust-vs-doubt slogans ('Same question? Same answer.', 'Stop vibe-coding your numbers. Start trusting them.') to make reliability the emotional throughline
