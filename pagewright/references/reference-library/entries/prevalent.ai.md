**Takeaways (roube como ideias, não copie):**
- Use uma headline serifada gigante de alto contraste com palavras-chave destacadas por cor ('asset', 'risk' no índigo de acento) para fazer uma afirmação de três orações ser escaneada como um manifesto
- Substitua a estética fria do nicho de segurança por uma ilustração narrativa full-bleed numa paleta apertada de 4 cores para parecer humana e memorável
- Combine uma display serifada editorial (Tiempos) com uma sans geométrica neutra (Matter) — gravidade no topo, legibilidade no corpo
- Estruture a copy como fragmentos staccato — nomeie três dores em três frases curtas, depois pivote para uma promessa de controle/empoderamento
