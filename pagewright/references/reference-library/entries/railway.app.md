Backdrop is a hand-painted starfield-and-clouds night sky — illustrative, not gradient-mesh — giving the dashboard a "ship at night" calm.
- **copy:** Calm, almost spiritual register for infra: hero reads "Ship software peacefully" — sells emotional relief, not features. Subhead names the product plainly ("the all-in-one intelligent cloud provider"); section H3s lean on the same reassurance-through-craft voice ("without the complexity", "without the growing pains").

**Takeaways (steal as ideas, don't copy):**
- Pair a literary serif headline (IBM Plex Serif) with a grotesque body (Inter) and a mono (JetBrains Mono) for in-product/log text — three voices that map to brand / UI / machine.
- Replace the default gradient-mesh hero with a hand-painted illustrated night sky; it gives 'technical-dark' warmth and atmosphere a CSS gradient can't.
- Crop a real, populated product console at the fold so it bleeds off-screen — implies a live system rather than a polished marketing mock.
- Reserve a single violet accent for one CTA, one inline link word, and live-status dots only — let the dark navy ground carry everything else.
