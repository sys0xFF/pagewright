**Takeaways (steal as ideas, don't copy):**
- Wire the UI to the product: a floating terminal window (CLI/API/CONSOLE tabs showing 'oxide auth login') connected by a literal line to the rack — proves the abstraction maps to real hardware.
- Use blueprint/figure framing ('FIG. 1', mono caption, faint grid) to borrow the credibility of an engineering spec sheet instead of marketing gloss.
- Let the product's own accent be physical — the brand green isn't a CSS variable, it's the glowing LEDs in the photo, so palette and hardware are the same color.
- Pair a humanist sans for the one big emotional claim with mono everywhere a machine speaks (nav, captions, terminal) to signal technical legitimacy.
