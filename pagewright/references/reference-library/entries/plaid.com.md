# Plaid — https://plaid.com

![plaid.com](../thumbs/plaid.com.jpg)

- **niche:** fintech
- **mood:** technical-dark
- **style:** gradient, cinematic, dark, colorful
- **palette:** bg `#0a4a8c` · ink `#ffffff` · accent `#7CFFB2` — hero headline 'data' / 'revolutionary' words rendered in mint-green, primary buttons, and the magenta-to-cyan gradient washing over the Ben Franklin engraving
- **type:** display *Bowery Street (high-contrast serif-ish display) paired with Plaid Sans for the second hero word* · body *Plaid Sans* — Editorial and confident — a tall, tight display face mixing serif and sans words within a single headline, giving an authoritative, almost magazine-cover voice over engineered sans body copy
- **sections:** hero › feature-ai-video-card › logos › feature-network › feature-ai-infrastructure › feature-network-quality › feature-bank-coverage-stats › showcase › cta › footer
- **signature:** A photorealistic engraved $100-bill Benjamin Franklin portrait, recolored in a hot magenta-to-cyan duotone gradient and holding a smartphone — money iconography literally pulled into the digital age, breaking fintech's usual sterile abstract-3D / dashboard-screenshot convention.
- **imagery:** Duotone-recolored currency engraving (Franklin) as a full-bleed hero illustration, fused with a real device shot; fine-line guilloche/engraving texture echoing banknote security patterns; deep blue gradient field with subtle radial line work. Imagery is metaphor-driven, not screenshot-driven.
- **copy:** Bold, declarative empowerment voice that promises transformation — hero: "Turn data into revolutionary financial products"; sub-headers lean witty ("Want access to 12K banks? We've got the API keys.").

**Takeaways (steal as ideas, don't copy):**
- Mix two type families WITHIN one headline (serif display + sans) and color only the verb/object words to make a single line read as both art and message.
- Recolor a culturally loaded real-world artifact (cash, a portrait) into a brand-gradient duotone instead of defaulting to abstract 3D fintech blobs.
- Let H2 section heads carry personality and wit ('We've got the API keys') rather than generic feature labels.
- Pair a saturated mint accent against deep ocean-blue so CTAs and key words pop without a hard color clash.
