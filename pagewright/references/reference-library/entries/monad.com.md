**Takeaways (steal as ideas, don't copy):**
- Pair a serious editorial serif headline with an all-monospace UI — the genre tension (literary vs terminal) signals 'rigorous but human' better than another all-sans dev page.
- Make the hero illustration the actual product topology: sources -> transform core -> destinations, with moving packet tags, so the value prop is understood before any copy is read.
- Use a desaturated sage green (not neon cyber-green) as the only color, applied sparingly to status pills and a single core glow — restraint reads as 'enterprise trust' in a security niche full of dark dashboards.
- Reserve a second color (blue) for exactly one CTA to create a clear primary action without a loud button system.
