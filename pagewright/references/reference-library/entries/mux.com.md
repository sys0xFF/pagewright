# Mux — https://mux.com

![mux.com](../thumbs/mux.com.jpg)

- **niche:** dev-tools (video infrastructure / API)
- **mood:** technical-dark
- **style:** mono-type, minimal, bold, illustrated
- **palette:** bg `#FAFAF8` · ink `#111111` · accent `#FF4A00` — Primary CTA pill (GET STARTED), the floating code/robot icons, secondary highlight orb on the talk-to-us button, and link/hover states; orange does almost all the work against a near-white off-white ground
- **type:** display *Geometric monospace (wide-tracked, condensed-tall caps — Mux's custom 'Mux Mono' / Akkurat Mono-style face)* · body *Humanist serif (transitional book serif for sub-head and running copy — an unexpected pairing against the mono display)* — Engineer-meets-editorial: the all-caps mono headline reads like a terminal banner while the serif body reads like a magazine, signalling 'serious infra, but made by people with taste'
- **sections:** nav › hero › logos › feature-grid › code-embed-demo › feature-infrastructure › feature-languages-sdks › testimonials › cta › footer
- **signature:** The headline is set in a wide all-caps MONOSPACE banner ("VIDEO FOR DEVELOPERS") but the supporting sentence drops into a warm humanist SERIF — a deliberate mono+serif clash that almost no dev-infra page dares, plus a hand-drawn cartoon robot mascot wedged into the hero where competitors would put a dashboard screenshot.
- **imagery:** Mostly type-driven on a faint engineering-graph-paper background (subtle grid lines bleed in at the edges). The only "image" up top is a quirky hand-illustrated orange-and-grey robot mascot ("Mux Robots"). Below the fold: real-world photographic video frames embedded inside rounded white cards with a floating </> code chip, plus monochrome wordmark logos (Fox, Bandcamp, Citizen, Glassdoor, Robinhood) in a bordered logo bar. Illustration + real footage instead of UI mockups.
- **copy:** Blunt, confident, developer-direct — names the audience instead of the product. Hero: "VIDEO FOR DEVELOPERS" with sub "Mux helps teams do more with video. Stream it, moderate it, search it, analyze it, transform it. Build cost-effectively in minutes, not months."

**Takeaways (steal as ideas, don't copy):**
- Pair an all-caps geometric MONOSPACE display headline with a humanist SERIF body — the friction between 'machine' and 'editorial' instantly reads premium and human in a dev category drowning in Inter.
- Lean on ONE hot accent (a single orange) against a near-white off-white ground and let it carry the CTA, icons, and mascot — restraint makes the one color feel loud.
- Inject personality with a hand-drawn mascot ('Mux Robots') in the hero instead of the obligatory product screenshot — humanizes hardcore infra without dumbing it down.
- Use a faint engineering graph-paper grid bleeding from the page edges as texture, not decoration — it signals 'built by engineers' subliminally without a single literal code block.
