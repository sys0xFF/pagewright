# Dovetail — https://dovetail.com

![dovetail.com](../thumbs/dovetail.com.jpg)

- **niche:** ai
- **mood:** technical-dark
- **style:** dark, 3d, mono-type, gradient
- **palette:** bg `#000000` · ink `#FFFFFF` · accent `#2C5FF6` — Electric-blue top ticker bar, the 3D pixel keycap object embedded in the headline, and link/hover states; everything else is near-monochrome so the blue reads as a single deliberate jolt
- **type:** display *Space Grotesk* · body *Inter* — Heavy geometric-grotesk headline set tight and oversized for a blunt, confident voice; Inter keeps body neutral and trustworthy; JetBrains Mono drives the uppercase ticker and micro-labels for a console/forensic feel, with PP Mondwest + EB Garamond held in reserve as retro-pixel and editorial accents
- **sections:** ticker-bar › nav › hero › logos › feature-built-for-trust › feature-drive-consistency › feature-privacy-without-friction › feature-pii-redaction › feature-ai-you-can-trust › feature-private-ai › cta › footer
- **signature:** A glossy 3D pixel-art keyboard keycap (with a peace-sign hand cursor on it) is wedged into the middle of the hero headline as a physical object replacing a word — turning flat type into a tactile, almost toy-like punchline that undercuts the usual sterile, all-text AI-platform hero.
- **imagery:** Almost no photography. A faint architectural blueprint grid recedes into the black void, anchored by one hyper-rendered 3D pixel keycap object as the single focal artifact. Brand logos sit as dim ghosted greys in the wall, plus tiny color app icons (Notion, Canva, AWS, Wise, Breville) tucked into the ticker bar. The treatment is forensic and constructed rather than lifestyle.
- **copy:** Blunt, contrarian declarative voice that picks a fight with hype — hero reads "Build with facts, not vibes" over a plain functional subhead about turning every customer signal into action.

**Takeaways (steal as ideas, don't copy):**
- Embed one rendered 3D object INTO the headline text as a word-replacement so the type itself becomes the hero art — no separate product shot needed.
- Run a thin electric-accent ticker bar across the very top with mono uppercase copy + tiny real app icons; it doubles as a promo banner and a 'we integrate with these' signal.
- Keep 95% of the page monochrome on pure black so a single saturated accent color carries all the energy — restraint makes the one blue feel intentional.
- Stack a deliberately wide type system (geometric-grotesk display + neutral sans body + mono labels + a retro pixel/serif accent) to feel both rigorous and playful instead of one flat sans everywhere.
