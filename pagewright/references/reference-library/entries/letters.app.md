# Letters — https://letters.app

![letters.app](../thumbs/letters.app.jpg)

- **niche:** ai (medical documentation / healthtech — AI letter-writing for clinicians)
- **mood:** clean-light
- **style:** gradient, minimal, illustrated
- **palette:** bg `#7E9FC4` · ink `#FFFFFF` · accent `#000000` — primary CTA pill ("Sign up for free" and nav "Sign up") rendered as a solid black rounded pill — the only hard, saturated element against the soft sky gradient
- **type:** display *Geist / Inter-like geometric grotesque (heavy weight)* · body *same grotesque sans, regular weight with selective bold spans* — confident and humane — ultra-bold tightly-tracked headline feels declarative, not clinical; bolded phrases in the subhead steer the eye like a spoken emphasis
- **sections:** hero › feature-before-after-demo › use-cases › features › pricing › our-doctors › cta › footer
- **signature:** A full-bleed daytime-sky gradient (periwinkle fading to pale blue) sprinkled with faint starlight dots — the page literally looks like calm open sky, the polar opposite of the sterile white-and-teal clinical SaaS convention. Healthcare software almost never lets the background carry the entire mood.
- **imagery:** A single hero \"Before / After\" diptych built from a paper-letter / envelope metaphor: a greyed, illegible handwritten scrawl card labeled \"Before\" slides behind a crisp, fully-typed clinical letter labeled \"After,\" both emerging from a folded white envelope shape. The product's value (messy → clean) is shown as a literal physical transformation rather than a UI screenshot or abstract 3D render.
- **copy:** Outcome-first, anti-bureaucracy voice that promises time back; hero reads "Patients, not paperwork" with a subhead emphasizing letters that "sound like you," "unlimited, gold-standard" output, and "Save 5+ hours weekly."

**Takeaways (steal as ideas, don't copy):**
- Let the gradient BE the design: a single calming sky wash can replace illustrations, photography, and color blocks while signaling the emotional benefit (calm, breathing room) better than feature copy.
- Demonstrate the product as a physical before/after artifact (handwritten scrawl morphing into a clean letter) instead of a UI screenshot — it reads instantly without the viewer parsing an interface.
- Use one and only one black pill CTA as the sole high-contrast element on an otherwise soft palette, so the eye has exactly one place to land.
- Pair an ultra-bold two-word headline ('Patients, not paperwork') with a subhead that bolds the load-bearing phrases — the emphasis spans do the persuading so the sentence can stay short.
