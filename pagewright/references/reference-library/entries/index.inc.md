**Takeaways (steal as ideas, don't copy):**
- Anchor a tool in an ecosystem: the subhead names Linear ('companion app for Linear') and a whole section is built around 'Access from anywhere with Raycast' — borrow trust by positioning against tools users already love.
- Warm-on-cool accent discipline: keep the entire palette cool (blue/violet) and reserve a single warm gradient (coral->amber) exclusively for the primary CTA so the eye has exactly one place to go.
- Sell the feeling, then prove it: hero = aspirational landscape with near-zero UI; the dense data table appears only as you scroll, separating desire from proof.
- Frame product UI in real OS chrome (macOS traffic lights, browser tabs, Share button) with believable sample data — it reads as a live tool, not a marketing mockup.
