**Takeaways (roube como ideias, não copie):**
- Demonstre o produto na dobra reconstruindo sua UI real em HTML/CSS em vez de jogar um screenshot — o layout ao vivo de três painéis com um balão de chat de agente em streaming vende fidelidade e velocidade num só relance.
- Desacople a cor de acento do CTA: mantenha o acento de marketing (índigo) como um pequeno sinal de sistema/status e faça o botão primário branco simples — lê mais como uma ferramenta, menos como um anúncio.
- Coloque o título do hero enorme com tracking muito apertado sobre um canvas quase-preto (#08090A, não #000 puro) e apoie-se num único sistema de painéis com borda finíssima para que cada seção de feature pareça uma janela para dentro do app.
- Use uma provocação de 'a categoria morreu' como um link inline secundário ('Issue tracking is dead → linear.app/next') para semear uma mudança narrativa sem entulhar a proposta de valor principal.
