**Takeaways (roube como ideias, não copie):**
- Construa um título com pesos mistos dentro de um mesmo grupo de palavras (fina + gorda) e recolora uma única palavra no destaque para criar ritmo sem adicionar elementos.
- Componha o tipo diretamente sobre uma foto de lifestyle humana full-bleed, escurecendo levemente a imagem para que o display branco continue legível.
- Use um rabisco de seta desenhada à mão apontando para o CTA secundário para injetar calor e guiar o olhar de um jeito que uma linha limpa jamais conseguiria.
- Mantenha um único destaque quente (amarelo-mel) fazendo tripla função — logo, realce do título, botão principal — para que a cor da marca leia como identidade, não decoração.
