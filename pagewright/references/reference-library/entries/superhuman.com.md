**Takeaways (roube como ideias, não copie):**
- Ilumine o sujeito do seu hero com sua cor de acento para que a fotografia e o fundo gradiente compartilhem uma única fonte de luz — a imagem se dissolve na tela em vez de ficar dentro de uma caixa.
- Construa o CTA principal como uma pílula de vidro escuro com um chip-de-seta de acento brilhante separado, rebitado na borda direita — o chip carrega a cor para que o botão permaneça calmo.
- Use uma grotesca condensada e alta em tamanho enorme para um hero de duas linhas; as formas de letra estreitas deixam uma frase longa preencher a largura e lê-se como 'engenheirada' em vez de decorativa.
- Enquadre uma suíte multi-produto como superpoderes nomeados (Mail, Docs, AI, cross-app) sob uma única frase-guarda-chuva aspiracional, para que um pacote se leia como uma única capacidade em vez de uma lista de features.
