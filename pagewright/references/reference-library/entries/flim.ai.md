# Flim — https://flim.ai

![flim.ai](../thumbs/flim.ai.jpg)

- **niche:** ai (creative search / reference engine for filmmakers, designers, agencies)
- **mood:** editorial-minimal
- **style:** mono-type, minimal, photographic, bento
- **palette:** bg `#FFFFFF` · ink `#0A0A0A` · accent `#00C853` — tiny play-triangle logo glyph at top-left, the status dot in the nav, and small interactive accents — deliberately sparse against an otherwise black-and-white page
- **type:** display *Helvetica Neue / Akzidenz-style grotesk, extra-bold, set enormous (the word 'Flim' spans nearly half the viewport)* · body *Monospaced grotesk for nav and UI labels (HOME, PRICING, SEARCH ⌘/), neo-grotesque sans for the hero subhead* — Swiss-poster confidence meets terminal/utility mono — gallery wall-text precision, not startup-friendly roundness
- **sections:** hero › logos › feature-find-influences › feature-search-create › feature-tools › feature-generate-edit › feature-collaborate › feature-save-share › footer
- **signature:** The hero IS the product demo: a live scattered grid of real cinematic reference stills floating on a faint blueprint grid, with an actual pre-filled search bar reading "KNIGHT ON A HORSE" — the page performs a visual search rather than describing one, breaking the AI-tool convention of leading with a chat box or abstract gradient.
- **imagery:** Cinematic photographic stills (movie frames, photography) presented as a loose moodboard / Pinterest-meets-grid collage at mixed sizes and rounded corners, scattered over a hairline graph-paper grid. The imagery is the content, not decoration — it sells the search-result payoff directly.
- **copy:** Declarative, agency-tagline cadence in stacked short lines — hero reads "The Creative Sidekick / Made for Agencies / Built for Storytellers." paired with section heads like "Search Beautifully. Create Purposefully."

**Takeaways (steal as ideas, don't copy):**
- Make the hero a working artifact: a real, pre-filled search box + live result grid turns 'show don't tell' into a literal demo above the fold.
- Pair an oversized neo-grotesque display word with monospace UI chrome — the type contrast alone reads as 'tool for serious creatives' without any illustration.
- Ration the accent: a near-monochrome page with ONE green glyph makes the color feel intentional and premium instead of decorative.
- Float photographic content on a faint blueprint/graph-paper grid to imply structure and 'searchability' under organic moodboard chaos.
