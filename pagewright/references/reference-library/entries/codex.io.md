# Codex — https://codex.io

![codex.io](../thumbs/codex.io.jpg)

- **niche:** data-infra (blockchain/crypto data API for dev-tools)
- **mood:** technical-dark
- **style:** dark, 3d, mono-type, bold
- **palette:** bg `#0A0A0A` · ink `#FFFFFF` · accent `#D4F838` — primary CTA fill ('Get Started' buttons), the floating 3D glass Codex cube, nav button highlight; deployed sparingly against near-black
- **type:** display *Grotesque/neo-grotesque condensed sans, all-caps (large oversized hero set tight)* · body *Humanist sans with slightly quirky letterforms (the 'a' and 'g' read warmer than the display)* — Loud, monumental, engineered — oversized condensed caps feel like a billboard/industrial stencil, softened by a friendlier body voice
- **sections:** hero › logos › feature-chart-data › feature-usd-pricing › feature-aggregate-data › logos › feature-network-support › testimonials › cta › footer
- **signature:** The lime-green photoreal 3D glass cube hovering over the hero, orbited by free-floating circular crypto/network token icons (Polygon, OP, etc.) scattered across the headline — a playful, almost toy-like physicality that breaks the flat, austere convention of blockchain-API marketing.
- **imagery:** Photoreal 3D rendered objects (a translucent glowing lime cube embossed 'CODEX') plus a constellation of circular orbs each holding a network/protocol glyph, floating over flat near-black. Partner logos rendered in muted monochrome grayscale rows. No screenshots or dashboards up top — pure object + type.
- **copy:** Confident superlative spec-claim, plain and direct — hero reads "POWER YOUR APP WITH THE FASTEST AND MOST RELIABLE BLOCKCHAIN DATA API" backed by hard numbers (70M+ tokens, 80+ networks).

**Takeaways (steal as ideas, don't copy):**
- Set the hero headline at near-window-width oversized condensed caps so the type itself is the hero visual — let the 3D object peek through/behind the letters instead of beside them.
- Use a single high-voltage accent (acid lime) only on CTAs and one hero object; keep everything else monochrome so the eye snaps to the action.
- Float small circular icon 'orbs' (network/integration logos) as scattered decorative satellites around the headline to signal breadth without a logo grid.
- Pair the loud display caps with a slightly warmer humanist body font and tiny inline feature pills (TRADING-READY / SUB-SECOND / ALL-IN-ONE) to deliver specs at a glance.
