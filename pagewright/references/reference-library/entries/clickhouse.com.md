**Takeaways (roube como ideias, não copie):**
- Racione um único destaque hipersaturado (chartreuse ácido) para exatamente três pontos de contato — barra superior, CTA principal, um rótulo eyebrow — e deixe o quase-preto + branco carregar todo o resto; a escassez faz a cor parecer uma luz de status do sistema, não decoração.
- Faça o título do hero absurdamente grande (uma laje equivalente a 4 linhas preenchendo metade do viewport) para que a página funcione como um outdoor com zero imagem — a tipografia É a direção de arte.
- Combine um subtítulo quantitativo duro ('millisecond queries at petabyte scale') com uma reivindicação suave de categoria ('leading database for AI') para obter tanto posicionamento emocional quanto a prova de um engenheiro em duas linhas.
- Use uma barra de anúncio chartreuse de largura total para divulgar novidades do produto (Claude-powered Agents) acima da navegação — transforma a borda superior num changelog sempre ligado que reforça o momentum.
