- **imagery:** Single-color (cornflower blue) outline illustration with subtle pastel pink/lavander glow accents on white. Loose, almost editorial linework — coding symbols (</>), cloud shapes, and padlocks woven into a human hand gesture. No photography, no UI screenshots, no 3D. Customer logos are simple greyscale wordmarks set in a clean grid with tiny blue heart marks.
- **copy:** Bold imperative promise with a contrarian twist — hero reads "Protect Everything You Build and Run", subhead lands the punch "AI alone doesn't make security faster. Context does."

**Takeaways (steal as ideas, don't copy):**
- Highlight ONE word of the headline in accent color (here 'Everything' in blue) to create a focal point without a separate eyebrow or label.
- End the subhead with a short, punchy two-clause contrarian line ('AI alone doesn't make security faster. Context does.') — bolding the payoff gives copy a spoken cadence.
- Replace the category-default dark dashboard hero with a warm single-line illustration to instantly differentiate in a fear-driven niche.
- Pair an inline email-capture field directly beside the primary CTA in the hero so the conversion ask sits at first glance, not below the fold.
