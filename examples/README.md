# Examples

Landing pages generated by Pagewright from a single-line brief. These are original products (not clones of
any reference), each with its full product UI built in HTML/CSS/SVG — no gray placeholders. Open any
`index.html` directly in a browser.

## Aegis — endpoint detection & response (EDR)

Brief: *an EDR platform for enterprise security teams; book a demo.* A dark "war-room" console: an animated
attack-path radar, a streaming detection feed with severity pills, and a threat-hunting query surface — all
hand-built.

![Aegis](aegis/screenshot.jpg)

## Cortex — product analytics

Brief: *a product-analytics platform for PM teams; start free.* A warm-light analyst's terminal: metric
cards, a conversion funnel that draws in on scroll, a retention cohort grid, and a trend line — all
hand-built in HTML/CSS/SVG.

![Cortex](cortex/screenshot.jpg)

## Relay — webhook & event delivery

Brief: *a developer platform for reliable webhook delivery; get an API key.* A graphite, industrial
"split-flap" event board where deliveries flip through statuses, plus a request-log table, a JSON payload
inspector, and a retry-backoff waterfall.

![Relay](relay/screenshot.jpg)
